Extended Ray Tracer by Nancy Ly

This webpage is dedicated to showcase my final project for the graphics course CS488 at the University of Waterloo. The purpose of this project is to further explore ray tracing techniques to improve the rendering of scenes. See below for a gallery of the different rendering techniques used.

  • Depth of Field
  • Texture Mapping

  • Animation

Depth of Field

The depth of field option and focal plane are specified in the lua file. Objects in focus will be clear, while objects further away from the focal point appear not in focus. There is currently a known issue where parts of objects appear clear despite not being in focus.

Image 2: Depth of field enabled with focal plane at origin

Texture Mapping

Textures on objects are specified in the lua file. Texture mapping can be applied to spheres and cubes.

Scene with texture mapping applied, follow by the texture maps

Method

I implemented this feature by following the method shown in class. In particular, given the texture coordinates uv, the corresponding point is found in the raster file. The color is interpolated from surrounding pixels using bilinear interpolation.

UV Mapping of a Cube

To determine the uv coordinates of a cube, I divided the texture file into 6 boxes (see image). Each box represents a face of the cube. I decided to map each box to the faces as shown in the image. Given a point on the cube, I determined which face the point lied on and calculated the uv coordinates using the size of the cube and the position of the point on the face.

Soft Shadows

The soft shadow option is specified in the lua file. Shadows in the scene will appear faded rather than crisp.

Image 1: Soft shadows disabled
Image 2: Soft shadows enabled

Glossy Reflections

The glossy reflection option specified in the lua file. Reflections will appear glossy, depending on the level of glossiness. Glossiness is determined by the object's Phong Exponent. An exponent of 1 means the object is diffuse. An exponent of over 100 means the object is very glossy to the point of mirror reflection. When glossy reflections are not enabled, all reflections will be mirror reflections.

Image 1: Glossy reflection disabled
Image 2: Glossy reflection enabled. The pink spheres become more glossy from left to right, with the first sphere being diffuse and the last sphere being very glossy.

Bump Mapping

Bump maps are specified in the lua file. Grayscale images are used to determine the bumps. Bump mapping can be applied to spheres and cubes.

Method

I implemented this feature by following the method shown in class. In particular, the normal vector is manipulated to give the effect of bumps on objects UV mapping of the objects are done the same way in texture mapping.

Anti-aliasing

The anti-aliasing option is specified in the lua file. Anti-aliasing allows the edges of the object to look less pixelated and more realistic.

Motion Blur

Objects in motion are specified in the lua file. The motion blur option is also specified in the lua file. Motion blur is used to show moving objects.

Animation

An animation was created through numerous lua files where the camera position is slightly altered each time. Each scene file rendered an image, with all of the images being strung together into a .gif through an external .gif creation application.

Final Scene

The final scene, titled Pac-Man's Galaxy. The scene uses texture mapping, mirror reflections in the ghosts, and anti-aliasing.
